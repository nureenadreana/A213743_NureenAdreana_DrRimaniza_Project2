## ReBite - Food Sharing App
**SDG 2: Zero Hunger**

## Description
ReBite is an Android app that connects people with surplus food 
to those who need it, reducing food waste in the community.

## Features
- 🏠 Browse free and reduced-price food listings
- 📍 GPS auto-fill pickup location
- 🔥 Firebase Authentication (Login/Register)
- ☁️ Firebase Firestore community feed (real-time)
- 🥗 Wger Nutrition API integration
- 💾 Room Database for bookings and bookmarks
- 🔖 Save/bookmark food items
- 📋 Manage your own listings

## Tech Stack
- Jetpack Compose
- MVVM + Repository Pattern
- Room Database
- Firebase Firestore + Authentication
- Retrofit (Wger API)
- GPS/Location Services
- Coil (Image Loading)
- Navigation Compose

## Setup
1. Clone the repo
2. Add your `google-services.json` in the `app/` folder
3. Build and run in Android Studio

SDG 2 - Zero Hunger: Reducing food waste by connecting 
food donors with recipients in the community.
